# For loop exercises

!!! question
    What is the problem with the following loops:

    ```javascript title="A problematic loop"
    for (i=0; i>10; i++){
        print(i);
    }
    ```

    ```javascript title="Another problematic loop"
    for (i=0; i<10; i--){
        print(i);
    }
    ```

    :warning: If you run them, you might have to kill your Fiji session.
